Tutorial

🎨 生图配置（Pollinations）

Pollinations 是本应用内置的 AI 生图服务，开启后聊天中的 AI 图片、查手机里的淘宝商品图、朋友圈图片都会自动生成。

① 获取 Key

进入 enter.pollinations.ai/sign-in 注册账号，登录后即可获得你的 API Key。没有 Key 无法使用。

② 在设置里填写

进入 Settings → 生图，打开「Pollinations生图」开关，将 Key 填入「Key」输入框，模型选择 flux（质量更好）或 turbo（速度更快）。

③ 在哪里能看到效果

开启后，聊天中 AI 回复里的 <IMAGE> 标签会自动生成图片；查手机→淘宝，点击商品详情时会生成商品图；查手机→微信→朋友圈，图片也会自动生成。

📱 查手机功能

查手机可以让你查看角色的手机内容，包括微信、支付宝、淘宝、地图、浏览器、贴吧、邮箱、网易云、设置，以及 AI 动态生成的第二页应用。

① 如何打开

点击主界面底部 Dock 栏的「📱 Peek」按钮，如果只有一个角色会直接打开，多个角色会弹出选择。

② 内容不是即时显示的

每个应用的内容都需要点击进入后手动生成，或者点击顶部「📦 批量生成」一次性生成多个。生成需要调用 AI，第一次会慢一点，请耐心等待。生成过的内容会缓存，下次打开不需要重新生成。

③ 小组件

第一页桌面顶部可以添加小组件（天气、日历、步数等）。点击「✏️ 编辑桌面」进入编辑模式，点击「➕ 添加小组件」选择类型。添加后点击小组件才会生成内容。

④ 第二页 AI 动态应用

向左滑动进入第二页，首次进入会看到一个「✨ 待生成」图标，点击后 AI 会根据角色人设生成 1-9 个专属应用。生成中图标显示 ⏳，失败显示 ❌ 可点击重试。

⑤ 浏览器批量生成

打开浏览器应用，在历史记录或书签页面右上角点击「多选」，勾选想要生成的页面，点击「批量生成」，一次 AI 调用即可生成全部页面，比逐个生成节省额度。

⑥ 网易云可以真实播放

网易云应用会搜索真实歌曲并播放，需要网络连接，部分歌曲因版权可能无法播放。支持上传本地歌词文件（.lrc / .srt / .vtt / .txt）。

⑦ 内容与角色挂钩

生成的内容基于角色人设和最近 50 条聊天记录，人设越详细、聊天越多，内容越贴合角色。建议先和角色多聊几句再使用查手机功能。

❗ 常见报错处理

AI 没有回复 / 一直转圈

检查 Settings → API配置，确认 API 地址和 Key 是否填写正确。可以点击「🔗 测试连接」验证。地址格式一般是 https://xxx.com，不需要带 /v1。

图片生成失败 / 图片不显示

确认生图开关已开启，且网络可以访问 gen.pollinations.ai。部分网络环境需要代理。图片加载失败会自动隐藏，不影响其他功能。

查手机生成失败

查手机生成失败通常是 API 超时或返回内容不符合 JSON 格式。点击页面上的「重试」按钮即可。如果反复失败，尝试换一个支持较长输出的模型。

屏幕底部出现红色错误条

这是 JS 报错提示。截图发给开发者，或者打开 Skin → 调试控制台，查看详细错误信息。

数据丢失

本应用数据存储在浏览器的 IndexedDB 中，清除浏览器数据或换浏览器会导致数据丢失。建议定期用角色卡导出功能备份重要角色。

🔗 副API设置

副API用于：图片识别、生成记忆、总结记忆等任务。
如果不填写，将使用主API。
建议使用支持Vision的模型。
